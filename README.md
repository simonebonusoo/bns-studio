# BNS Studio — website (React + Tailwind + Framer Motion)

## Avvio rapido
```bash
npm i
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Dove cambiare contenuti
- `src/sections/*` (testi, servizi, portfolio, prezzi, risorse)
- `src/components/Navbar.tsx` (menu)
- `src/styles.css` (effetti: griglia, glass, noise)
